## Структура проекта

- [`schema.js`](./mongo/schema.js) - скрипт с инициализацией валидационных схем и индексов для базы данных.
- [`validation.js`](./mongo/validation.js) - скрипт с демонстрацией валидации схем.
- [`data.js`](./mongo/data.js) - скрипт для вставки тестовых данных.
- [`queries.js`](./mongo/queries.js) - скрипт с функциями, представляющими CRUD операции, использующиеся в проекте.
- [`schema_design.md`](./mongo/schema_design.md) - описание проектирования документной модели.
- [`api_gateway`](./api_gateway/) - проект из Лабораторной работы №2 с добавленной MongoDB базой данных.
