# Document searcher

Сервис для поиска документов в индексе

## Конфигурация Docker образа

Docker-образ поддерживает следующие параметры конфигурации

### Переменные окружения

- `PORT` - порт, который будет прослушивать сервис.
- `MEILISEARCH_URL` - URL до Meilisearch, к которому будет обращаться сервис для поиска документов.
- `MEILISEARCH_API_KEY` - ключ для API от Meilisearch, по которому будут отправляться запросы.
- `INDEX_NAME` - название индекса, в котором сервис будет искать документы.
